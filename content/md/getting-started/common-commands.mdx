# Common Development Commands

## Project Setup

### Installation
```bash
# Install dependencies
bun install

# Install specific package
bun add package-name

# Install dev dependency
bun add -d package-name
```

### Environment Setup
```bash
# Copy environment variables
cp .env.example .env

# Generate Prisma client
bunx prisma generate

# Push database changes
bunx prisma db push
```

## Development

### Starting Servers
```bash
# Start development server
bun dev

# Start development server with specific port
bun dev -p 3001

# Start production server
bun start
```

### Database Commands
```bash
# Create migration
bunx prisma migrate dev --name migration-name

# Apply migrations
bunx prisma migrate deploy

# Reset database
bunx prisma migrate reset

# Open Prisma Studio
bunx prisma studio
```

## Testing

### Running Tests
```bash
# Run all tests
bun test

# Run tests in watch mode
bun test --watch

# Run specific test file
bun test path/to/test.ts

# Run tests with coverage
bun test --coverage
```

### Linting
```bash
# Run linter
bun lint

# Fix linting issues
bun lint --fix

# Check types
bun type-check
```

## Building

### Build Commands
```bash
# Build for production
bun run build

# Build with specific configuration
bun run build --config custom.config.js

# Analyze build
bun run build:analyze
```

### Deployment
```bash
# Deploy to staging
bun run deploy:staging

# Deploy to production
bun run deploy:prod

# Preview deployment
bun run deploy:preview
```

## Git Commands

### Basic Git
```bash
# Create new branch
git checkout -b branch-name

# Switch branch
git checkout branch-name

# Pull latest changes
git pull origin main

# Push changes
git push origin branch-name
```

### Git Workflow
```bash
# Stage changes
git add .

# Commit changes
git commit -m "commit message"

# Stash changes
git stash

# Apply stashed changes
git stash pop
```

## Maintenance

### Cleanup
```bash
# Remove node_modules
rm -rf node_modules

# Clear cache
bun cache clean

# Remove build files
rm -rf .next
```

### Updates
```bash
# Update dependencies
bun update

# Update specific package
bun update package-name

# Check for outdated packages
bun outdated
```

## Debugging

### Debug Commands
```bash
# Start debugger
bun debug

# Run with debug logs
DEBUG=* bun dev

# Profile application
bun profile
```

### Logging
```bash
# View logs
bun logs

# Clear logs
bun logs clear

# Follow logs
bun logs follow
```

## Security

### Security Checks
