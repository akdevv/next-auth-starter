## Getting Started

1. **Installation**
   ```bash
   git clone https://github.com/yourusername/next-auth-starter.git
   cd next-auth-starter
   bun install
   ```

2. **Development**
   ```bash
   bun dev
   ```

3. **Building**
   ```bash
   bun run build
   ```
